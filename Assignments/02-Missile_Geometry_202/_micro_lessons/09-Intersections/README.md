# Module 09 — Intersections

Detecting when paths cross boundaries: from two-segment math to full country-airspace analysis across real GeoJSON datasets.

| # | Notebook | Description |
|---|---|---|
| 00 | [Lines as Paths](./00-Lines_as_Paths.ipynb) | Representing missile trajectories as GeoJSON LineStrings; builds `data/paths.geojson` |
| 01 | [Line Segment Intersection](./01-Line_Segment_Intersection.ipynb) | Detecting whether two segments cross using orientation and cross-product math |
| 02 | [Line vs Polygon Basics](./02-Line_vs_Polygon_Basics.ipynb) | Testing a path against a polygon by checking each edge pair |
| 03 | [Detecting Intersections](./03-Detecting_Intersections.ipynb) | Running all paths against real country boundary data |
| 04 | [Highlighting Intersected Features](./04-Highlighting_Intersected_Features.ipynb) | Coloring hit vs. miss countries on a map; multi-path aggregate view |
| 05 | [Applications: Missile Paths](./05-Applications_Missile_Paths.ipynb) | Full path report — airspace crossings, range, bearing, and shared overflights |
